# Hospital Management System

## 📌 Project Overview
A Hospital Management System built using FastAPI, SQLAlchemy, and SQLite.

## 🚀 Features
- User Authentication (JWT)
- Doctor Management
- Patient Management
- Appointment Management
- CRUD Operations
- REST API
- Swagger API Documentation

## 🛠 Technologies Used
- Python
- FastAPI
- SQLAlchemy
- SQLite
- Pydantic
- Uvicorn

## 📂 Project Structure
```
Hospital-Management-System/
│── main.py
│── database.py
│── models.py
│── schemas.py
│── crud.py
│── auth.py
│── requirements.txt
│── hospital.db
│── routers/
```

## ▶️ Run the Project

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Open in your browser:

```
http://127.0.0.1:8000/docs
```
